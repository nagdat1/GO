# كيفية الحصول على Chat ID لمجموعة Telegram

## الطريقة الأولى: استخدام Bot @userinfobot

1. أضف البوت `@userinfobot` إلى مجموعتك
2. أرسل رسالة في المجموعة (أي رسالة)
3. سيُرسل لك البوت معلومات المجموعة، بما في ذلك **Chat ID**
4. Chat ID للمجموعة يكون سالباً (مثل: `-1001234567890`)

## الطريقة الثانية: استخدام Bot @RawDataBot

1. أضف البوت `@RawDataBot` إلى مجموعتك
2. أرسل رسالة في المجموعة
3. سيرسل لك البوت بيانات JSON، ابحث عن `"chat":{"id":-1001234567890}`
4. الرقم بعد `"id":` هو Chat ID الخاص بمجموعتك

## الطريقة الثالثة: استخدام Webhook URL مع Chat ID

إذا كنت تستخدم Webhook URL مع Chat ID في الرابط:
```
https://go-production-e51a.up.railway.app/personal/-1001234567890/webhook
```

استبدل `-1001234567890` بـ Chat ID الخاص بمجموعتك.

## الطريقة الرابعة: تحديث config.py

يمكنك تحديث `TELEGRAM_CHAT_ID` في `config.py`:
```python
TELEGRAM_CHAT_ID = os.getenv('TELEGRAM_CHAT_ID', '-1001234567890')
```

⚠️ **ملاحظة مهمة**: Chat ID للمجموعة يكون سالباً!

## ملاحظات مهمة:

1. ✅ تأكد من أن البوت عضو في المجموعة
2. ✅ تأكد من أن البوت لديه صلاحيات إرسال الرسائل
3. ✅ Chat ID للمجموعة يكون سالباً (يبدأ بـ `-`)
4. ✅ Chat ID الشخصي يكون موجباً (مثل: `8169000394`)

## اختبار البوت:

بعد الحصول على Chat ID، يمكنك اختبار البوت بإرسال رسالة تجريبية:
- افتح الرابط: `https://api.telegram.org/bot<YOUR_BOT_TOKEN>/sendMessage?chat_id=-1001234567890&text=Test`
- استبدل `<YOUR_BOT_TOKEN>` بـ Token البوت الخاص بك
- استبدل `-1001234567890` بـ Chat ID مجموعتك

إذا نجح الاختبار، سترى رسالة "Test" في المجموعة.
