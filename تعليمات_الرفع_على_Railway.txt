الخطوة 1: إنشاء حساب Railway
───────────────────────────────────────────────────────────────────────────
   1. اذهب إلى: https://railway.app
   2. اضغط Login
   3. سجل بحساب GitHub (الأسهل والأسرع)

الخطوة 2: إنشاء Repository في GitHub
───────────────────────────────────────────────────────────────────────────
   1. اذهب إلى: https://github.com
   2. اضغط New Repository
   3. اكتب اسم (مثال: tradingview-telegram-bot)
   4. اختر Public أو Private
   5. اضغط Create repository

الخطوة 3: رفع الملفات على GitHub
───────────────────────────────────────────────────────────────────────────
   1. في الـ Repository الجديد، اضغط Upload files
   2. ارفع هذه الملفات:
      ✅ main.py
      ✅ requirements.txt
      ✅ Procfile
      ✅ railway.json (اختياري)
   3. اضغط Commit changes

الخطوة 4: ربط Railway مع GitHub
───────────────────────────────────────────────────────────────────────────
   1. في Railway Dashboard، اضغط New Project
   2. اختر Deploy from GitHub repo
   3. اختر الـ Repository الذي أنشأته
   4. Railway سيكتشف تلقائياً أنه Python project
   5. سيبدأ البناء تلقائياً (انتظر 2-3 دقائق)

الخطوة 5: الحصول على URL
───────────────────────────────────────────────────────────────────────────
   1. بعد النشر، اضغط على المشروع
   2. في Settings → Domains
   3. أو في Deployments → View Logs
   4. ستجد Public URL (مثل: https://your-app.railway.app)
   5. انسخ هذا الرابط

الخطوة 6: إعداد Alert في TradingView
───────────────────────────────────────────────────────────────────────────
   1. في TradingView:
      - اضغط Alert (🔔) → Create Alert
      - اختر الاستراتيجية: nagdat
      - فعّل Webhook URL
      - ضع الرابط: https://your-app.railway.app/webhook
      - في Message:
        {{strategy.order.comment}}
        {{ticker}} - {{close}}
        {{time}}
      - اضغط Create

الخطوة 7: الاختبار
───────────────────────────────────────────────────────────────────────────
   1. افتح في المتصفح:
      https://your-app.railway.app/test
   2. يجب أن تستلم رسالة في Telegram ✅
   3. أو انتظر إشارة من الاستراتيجية
